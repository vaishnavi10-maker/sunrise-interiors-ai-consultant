# sunrise-interiors-ai-consultant

An AI-powered multilingual voice consultant that conducts live customer calls for interior design consultations.

## 🏗️ Architecture

User
  ↓
Frontend Application
  ↓
FastAPI Backend
  ↓
Bolna Voice Agent
  ↓
Deepgram Speech-to-Text
  ↓
LLM Conversation Engine
  ↓
ElevenLabs Text-to-Speech
  ↓
Live Phone Call
